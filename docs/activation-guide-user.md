# 工具激活指南

## 获取激活码

购买工具后，您将收到一组专属激活码，格式如下：

```
XXXX-XXXX-XXXX-XXXX
```

示例：`A7KN-3PBR-WDXY-8HM4`

---

## 激活步骤

### 第一步：登录账户

访问网站，使用手机号或邮箱登录您的账户。如果还没有账户，请先注册。

### 第二步：进入用户中心

登录后，点击右上角头像 →「用户中心」，或直接访问用户中心页面。

### 第三步：输入激活码

在「激活工具」区域，输入您收到的激活码，点击「激活」按钮。

### 第四步：下载使用

激活成功后，工具会出现在「已激活工具」列表中，点击「下载」按钮即可获取安装包。

---

## 常见问题

**Q：激活码不区分大小写吗？**
A：是的，输入大写或小写都可以，系统会自动识别。

**Q：一个激活码可以用几次？**
A：每个激活码仅限使用一次，激活后即绑定您的账户。

**Q：激活码可以转让给别人吗？**
A：未使用的激活码可以转给他人使用，但一旦激活就绑定账户，无法转移。

**Q：输入激活码提示失败怎么办？**
A：请检查以下几点：
- 确认激活码输入正确，注意区分相似字符
- 确认激活码尚未被使用
- 确认您已登录账户
- 如仍有问题，请联系客服

**Q：激活后在哪里下载？**
A：在用户中心的「已激活工具」列表中，每个工具右侧都有下载按钮。
